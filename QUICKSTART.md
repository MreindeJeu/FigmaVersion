# ⚡ Quick Start - 3 Steps

## 1️⃣ Initialize Data
```bash
npm run init-data
```

## 2️⃣ Start Server
```bash
npm run server
```

## 3️⃣ Open App
Open Figma Make preview - Done! 🎉

---

## What Now?

### View Your Data
```bash
npm run list-data
```

### Edit Data Two Ways:

**A) Admin Panel**
- Go to `/admin` in the app
- Click "NEW DEPLOYMENT", "NEW PILOT", etc.

**B) Edit Files Directly**
- Open `/data/pilots/P001.json` in your editor
- Make changes and save
- Refresh the app

### Add New Entries
```bash
# Copy a template
cp templates/pilot-template.json data/pilots/P999.json

# Edit it, then refresh the app!
```
